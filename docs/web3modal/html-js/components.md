# Components

## `<w3m-core-button>`

Pre-styled connect/disconnect web-component button that reacts to modal's theme changes.

```html
<body>
  <w3m-core-button></w3m-core-button>
</body>
```

Following attributes can be set

```ts
interface Attributes {
  icon?: "show" | "hide";
  label?: string;
}
```
