# About Web3Inbox 

Web3Inbox is a widget UI library built on top of [WalletConnect](https://walletconnect.com) and
powered by Web3Inbox.

It is designed to make complex tasks like subscribing to get push notifications, displaying
notification history while being completely themeable.
