# Styling 

Web3Inbox Widget is fully themeable using plain css. Simply host a `web3inbox.css` file on the roof
of the specified origin. (I.E `dappOrigin`) and the styles will be applied. 

## Colors

The following colors are used throughout the entire web app, and can all be themed.

For example, this turns everything white.

```css 
:root {
	--bg-color-1: '#ffffff'
	--bg-color-2: '#ffffff'
	--bg-color-3: '#ffffff'
	--bg-color-4: '#ffffff'
	--bg-gradient-1: '#ffffff'
	--bg-gradient-2: '#ffffff'
	--active-link-gradient: '#ffffff'
	--sender-bubble-bg: '#ffffff'
	--sender-box-shadow: '#ffffff'
	--fg-color-1: '#ffffff'
	--fg-color-2: '#ffffff'
	--fg-color-3: '#ffffff'
	--fg-color-4: '#ffffff'
	--fg-color-5: '#ffffff'
	--border-color-1: '#ffffff'
	--border-color-2: '#ffffff'
	--accent-color-1: '#ffffff'
	--error-color-1: '#ffffff'
	--icon-color-1: '#ffffff'
	--qr-color-1: '#ffffff'
	--brightness-multiplier: '#ffffff'
	--shimmer-color-1: '#ffffff'
	--shimmer-color-2: '#ffffff'
}
```

## Advanced 

One can also modify every element's look and feel to match their own theme.

For example, this makes all buttons square.

```css

.Button {
	border-radius: 0;
}
```
