## Introduction

Web3Inbox has two distinctive branches of SDKs.

### Web3Inbox SDK: Wallet Development

The Web3Inbox SDK allows one to easily make use of WalletConnect's [push](../specs/clients/push/wallet-client-api.md) and [chat](../specs/clients/chat/client-api.md) SDKs. This is done by rendering a WebView of the Web3Inbox dapp.

#### Features

Some of the key features of the Web3Inbox SDK include:

- **Manage what SDKs to use and UI pages to show**
- **Display push notification history**
- **Chat with peers**

### Web3Inbox Widget: Dapp Developement
The Web3Inbox Widget utilizes `iframe`s to display  Web3Inbox in a widget friendly view. Currently making use of WalletConnect's [push](../specs/clients/push/wallet-client-api.md) to subscribe and display notifications.

#### Features
Some of the key features of the Web3Inbox Widget include:

- **Inject the account connected to your dapp**
- **Theme how it looks with vanilla css**
- **Subscribe to your dapp and display notifications**
