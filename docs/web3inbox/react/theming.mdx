# Theming

Web3Inbox Widget is fully themeable using plain css. Simply host a `web3inbox.css` file on the roof
of the specified origin. (I.E `dappOrigin`) and the styles will be applied. 

## Colors

The following colors are used throughout the entire web app, and can all be themed.

For example, this keeps everything default.

```css

	--bg-color-1: 'hsla(0, 0%, 8%, 1)'
	--bg-color-2: 'hsla(180, 4%, 16%, 1)'
	--bg-color-3: 'hsla(0, 0%, 0%, 0.1)'
	--bg-color-4: 'hsla(0, 0%, 100%, 0.1)'
```
