## Get your first push notification

Getting your first push notification is straightforward:

1. Subscribe using your preferred client's subscribe method
2. Run the following `cURL`:

```bash
curl --location 'https://dev.notify.walletconnect.com/<YOUR_PROJECT_ID>/notify' \
--header 'Accept: */*' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer <YOUR_PROJECT_SECRET>' \
--data '{
    "notification": {
        "body": "Push Notification Body",
        "title": "Push Notification Title",
        "icon": "https://push.notification.image/icon.png",
        "url": "https://your.project.url",
        "type": "notification_type"
    },
    "accounts": [
        "eip155:1:<YOUR_EIP155_ACCOUNT>"
    ]
}'
```
